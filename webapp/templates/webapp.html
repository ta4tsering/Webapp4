How to use this WebApp

Requirements before using the webapp

-: Workid of the pecha
-: User Token to use the webapp
-: Pictures of the pecha

Steps to do it

1. First get the workid of the pecha.
2. Put the User Token to use this webapp.
3. Then get the pictures of the pecha for to run OCR.
4. choose the OCR engine from Google-OCR or Namsel-OCR.
5. Press the "Run the OCR" button.
