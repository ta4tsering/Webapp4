How to use this WebApp

Requirements before using the webapp

-: Workid of the pecha or File of the pecha workid
-: User Token to use the webapp
-: Which OCR Engine to use

Steps to do it

1. you either get the workid of the pecha or get the File of the workid for.
2. Put the workid or upload the workid file
3. Then put the User Token to use this webapp.
4. Choose the OCR engine from Google-OCR or Namsel-OCR.
5. Press the "Run the OCR" button.
